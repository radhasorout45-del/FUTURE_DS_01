📊 Supermarket Sales Insights Dashboard
📌 Project Overview

This project presents a Sales Insights Dashboard built to analyze supermarket data and extract meaningful business insights. The dashboard helps in understanding sales performance, customer behavior, and product trends.

🎯 Objectives
Analyze overall sales performance
Identify top-performing product categories
Understand customer distribution by gender
Study sales trends across different cities
Analyze payment method usage
📁 Dataset Information

The dataset contains supermarket sales records including:

Product Line
City
Gender
Quantity
Gross Income
Payment Method
Ratings
📊 Dashboard Features
🔹 Key Metrics
Total Sales: 323K
Average Rating: 6.97
Total Orders: 1K
🔹 Visualizations
📈 Quantity Trend Analysis – Shows how product demand changes over time
💰 Gross Income Analysis – Highlights income distribution
🥧 Payment Method Distribution – E-wallet, Cash, Credit Card usage
🍩 Product Line Distribution – Sales contribution by categories
👥 Customer Distribution by Gender
🌍 Sales by City – Yangon, Mandalay, Naypyitaw
🛠️ Tools & Technologies Used
Power BI / Tableau (whichever you used, replace accordingly)
Microsoft Excel (for data preprocessing)

📈 Key Insights
E-wallet and Cash payments are almost equally preferred
Sales are highest in Naypyitaw
Customer distribution is nearly balanced between male and female
Certain product lines contribute more significantly to total revenue
🚀 How to Use
Download the dataset
Open the dashboard file in Power BI / Tableau
Use filters (City, Gender, Product Line) to explore insights
